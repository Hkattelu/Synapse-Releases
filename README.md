## ğŸ›  Installation

### Windows
1. Download the `Synapse-X.X.X-win-x64.exe` installer.
2. Run the installer. Synapse will launch automatically after installation.

### macOS (Supports M1/M2/M3 and Intel)
1. Download the `Synapse-X.X.X-mac-universal.dmg`.
2. Open the DMG and drag **Synapse** to your Applications folder.
3. *Note*: Since this is a community-driven project, you may need to right-click and select "Open" the first time to bypass Apple's Gatekeeper for unsigned apps.

### Linux
1. Download the `Synapse-X.X.X-linux-x64.AppImage`.
2. Make the file executable: `chmod +x Synapse-X.X.X-linux-x64.AppImage`.
3. Run the AppImage.
